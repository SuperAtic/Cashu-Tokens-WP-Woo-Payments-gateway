== Description ==

**Cashu Woocommerce Payment Gateway** is a secure and easy-to-use plugin that seamlessly integrates the CASHU payment gateway with WooCommerce. This plugin enables your customers to make payments using ecash generated on the CASHU protocol during the WooCommerce checkout process.

**Key Features:**
- Accept CASHU tokens as payments on any WooCommerce store.
- Compatibility with certified wallets like [CashuMe](https://cashu.me) and [NustStash](https://alpha.nutstash.app/) web wallets.
- Support for [browser extensions](https://github.com/SuperAtic/NutStash-Browser-Extension) and the mobile application [eNuts](https://github.com/cashubtc/eNuts).

For more information about CASHU, please visit the official [CASHU website](https://cashu.space).

== Installation ==

1. Upload the `cashu-woocommerce-payment-gateway` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in your WordPress admin panel.
3. Navigate to WooCommerce > Settings > Payments and activate the **Cashu Payment Gateway**.

== Frequently Asked Questions ==

**Q:** Do you have answers? 
**A:** Yes, if you have questions.

**Q:** What is CASHU?
**A:** CASHU is a digital payment platform that allows users to generate and use ecash for online payments. It provides a secure and private way to make online transactions.

**Q:** How do I set up CASHU for my WooCommerce store?
**A:** To set up CASHU for your WooCommerce store, follow these steps:
1. Install and activate the Cashu Woocommerce Payment Gateway plugin.
2. Configure your CASHU API credentials in the plugin settings.
3. Enable the Cashu Payment Gateway in WooCommerce settings.
4. Customers can now select CASHU as a payment option during checkout.

**Q:** Is CASHU Woocommerce Payment Gateway compatible with mobile wallets?
**A:** Yes, this plugin is compatible with mobile wallets such as CashuMe and eNuts, making it convenient for users to pay using their mobile devices.
